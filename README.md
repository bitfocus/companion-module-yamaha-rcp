## Yamaha Remote Control Protocol

**Available for the following Yamaha Pro Audio Devices**

* Rivage PM
* CL1/3/5
* QL1/5
* TF1/3/5
* TF-Rack

**Available commands**

* Selected "set" commands
* Recall Scenes
